# Explain (только ответ, без изменений)

**Режим:** только ответ на вопрос пользователя. Ничего не менять в проекте.

- Не создавать, не редактировать и не удалять файлы.
- Не запускать команды с побочными эффектами (сборка, установка, пуш в git и т.п.).
- Отвечать кратко и по существу на вопрос пользователя: объяснение, ссылки на код/файлы, рекомендации текстом.

Если для ответа нужно прочитать файлы — только читать (read), без правок.
